MSUSEL Software Injector

The msusel software injector is designed to provide the basis for conducting empirical studies concerning software design disharmonies. This is done by providing simple transforms which allow the injection of simple artifacts into existing software source files. The combination of these simple transforms can allow the creation of much more complex artifacts such as code smells, anti-patterns, design patterns, code issues, and design pattern grime.

Developing these complex transformations, or injection strategies, allows for the capability to evaluate their effect on measurable qualities such as maintainability, security, reliability, and even technical debt.
